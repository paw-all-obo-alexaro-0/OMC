# OMCtp
OMCtp is a game based on blocks, similar to other sandbox games like Minecraft (Mojang Studios), Luanti, Hytale (Hypixel Studios), and many more. It focuses on technology, magic, consistent physics and theory, and a large variety of blocks, items, and more. The game is currently in early development, and there is no release date yet.  
tp in OMCtp stands for "third person", and the game is played in third person perspective.

## Development
OMC is written in C++ and uses OpenGL for rendering. For further information, please inspect the source code, which is available on GitHub.
